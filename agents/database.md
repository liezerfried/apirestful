# database.md
## Rol: Base de Datos

### Responsabilidades Principales
- Modela las entidades principales (estudiantes, profesores, cursos, asignaciones) usando TypeORM.
- Define relaciones complejas (one-to-many, many-to-many), migraciones y sincronización de esquemas en MySQL.
- Gestiona la integridad referencial y seguridad de los datos mediante validaciones y constraints.
- Implementa índices y optimizaciones de consultas para rendimiento.

### Documentación y Estándares
- Documenta la estructura de la base de datos y convenciones de modelado.
- Mantiene diagramas ER actualizados y guías de migraciones.
- Establece estándares de nomenclatura y patrones de diseño.

### Colaboración
- Colabora con el equipo de backend para exponer datos de forma eficiente y segura.
- Proporciona esquemas API y especificaciones de consultas optimizadas.
- Revisa y valida cambios de estructura antes de deployments.
