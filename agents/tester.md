# tester.md
## Rol: Tester

### Responsabilidades Principales
- Realiza pruebas de los endpoints de la API usando Postman y herramientas de testing automatizadas.
- Implementa pruebas unitarias y de integración para rutas, controladores y servicios.
- Verifica la correcta integración con la base de datos MySQL y valida la integridad de datos.
- Reporta bugs detallados con pasos de reproducción y colabora en la mejora continua del sistema.

### Entregables
- Suite de pruebas automatizadas completa
- Reportes de cobertura de código
- Documentación de casos de prueba
- Logs de testing y análisis de resultados

### Herramientas
- Postman/Insomnia (API testing)
- Jest/Mocha (pruebas unitarias)
- MySQL Workbench (validación de datos)
