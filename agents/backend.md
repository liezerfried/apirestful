# backend.md
## Rol: Backend

### Descripción General
Responsable del desarrollo de la API RESTful que sirve como intermediaria entre la aplicación frontend y la base de datos. Implementa la lógica de negocio, validaciones y autenticación del sistema.

### Responsabilidades Principales
- Implementa la API RESTful usando Node.js, Express y TypeScript.
- Define rutas, controladores y servicios para estudiantes y profesores.
- Integra la base de datos MySQL usando TypeORM para el modelado de entidades y operaciones CRUD.
- Gestiona la validación de datos, autenticación y manejo de errores.
- Colabora con el agente database.md para el modelado, migraciones y optimización de consultas.
- Expone endpoints claros y documentados para pruebas con Postman.
- Implementa middleware para seguridad, logging y manejo de excepciones.
- Garantiza el cumplimiento de estándares REST y buenas prácticas de código.

### Tecnologías Principales
- Node.js + Express
- TypeScript
- TypeORM
- MySQL

### Dependencias de Agentes
- database.md (esquema y migraciones)
