# architect.md
## Rol: Arquitecto

### Responsabilidades Principales

- **Diseño de Arquitectura**: Define la arquitectura general del sistema, incluyendo estructura de carpetas, separación de capas (rutas, controladores, servicios, modelos) y patrones de comunicación.
- **Selección Tecnológica**: Evalúa y selecciona las tecnologías principales del stack (Node.js, Express, TypeScript, TypeORM, MySQL) basándose en requisitos del proyecto.
- **Patrones y Convenciones**: Establece patrones de diseño (MVC, Repository, Dependency Injection) y convenciones de código para garantizar consistencia en todo el proyecto.
- **Documentación de Módulos**: Documenta la interacción entre componentes internos (API endpoints, flujos de base de datos, integración de pruebas) para facilitar el onboarding del equipo.
- **Escalabilidad y Mantenibilidad**: Garantiza que la arquitectura soporte crecimiento futuro, sea fácil de mantener y permita cambios sin afectar otros módulos.

### Entregables

- Diagrama de arquitectura del sistema
- Especificación de estructura de carpetas
- Guía de convenciones y estándares de código
- Documentación de APIs y flujos de datos
