# frontend.md
## Rol: Frontend

### Responsabilidades
- Consumir la API RESTful del backend siguiendo los contratos definidos
- Implementar una interfaz de usuario intuitiva y responsiva
- Gestionar el estado de la aplicación y la comunicación con el servidor
- Validar datos en el cliente antes de enviarlos al backend

### Tecnologías y Herramientas
- Framework frontend (React, Vue, Angular, etc.)
- Herramientas HTTP (Fetch API, Axios)
- Gestión de estado (Redux, Pinia, Context API)
- Testing (Jest, Vitest, Cypress)

### Colaboración
- Colabora activamente con backend para definir y validar contratos de API
- Participa en sprints de especificación de endpoints
- Reporta inconsistencias o mejoras en la API

### Ejemplos de Consumo
- Ejemplos de solicitudes GET, POST, PUT, DELETE
- Manejo de errores y respuestas
- Implementación de autenticación y autorización
- Paginación y filtrado de datos

### Consideraciones
- Performance y optimización de renderizado
- Manejo de loading, error y success states
- Documentación de componentes y funciones
