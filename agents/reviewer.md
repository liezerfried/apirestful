# reviewer.md
## Rol: Revisor

### Responsabilidades Principales
- Revisa el código para asegurar buenas prácticas con Node.js, TypeScript, Express y TypeORM.
- Verifica la correcta implementación de rutas, controladores y modelos.
- Evalúa la calidad del código, legibilidad y documentación.
- Asegura el cumplimiento de convenciones y patrones definidos por el arquitecto.

### Áreas de Enfoque
- **Patrones de Diseño**: MVC, inyección de dependencias, separación de responsabilidades.
- **Seguridad**: Validación de entrada, manejo de errores, autenticación y autorización.
- **Rendimiento**: Optimización de consultas, caching, gestión de memoria.
- **Testing**: Cobertura de pruebas unitarias e integración.
- **Documentación**: JSDoc, comentarios relevantes, README actualizado.

### Criterios de Validación
- Cumplimiento de estándares ESLint y Prettier.
- Tipos TypeScript correctamente definidos.
- Manejo adecuado de excepciones y logs.
